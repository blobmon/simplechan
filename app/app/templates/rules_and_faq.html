Rules and FAQs :

Rules :

  • Do not spam and try to keep the discussion on-topic.
  • Do not post unsafe or inappropriate/illegal content.
  • Try to keep the posts qualitative and/or informative.

Frequently Asked Questions :

Is name necessary while making posts?
No. name field is optional and can be left blank.

Is attaching image necessary while making posts?
Attaching image is only necessary if you are starting a thread. Otherwise, it is optional.

How do I reply to a post?
Just click on the post number whom you wish to reply to. This will bring up the reply dialog.

What are the supported file types that can be uploaded?
Currently supported file types are jpg/jpeg, png, gif for images and webm for videos

What is the maximum size of file that can be uploaded?
Maximum size of uploadable file is approximately 5MB.
